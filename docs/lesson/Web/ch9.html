5、使用EasyUI之前必须现象生命UI控件，有哪两种声明方式？

EasyUI提供了两种主要的声明方式：
HTML标记声明方式：
使用HTML标记直接在页面中声明EasyUI控件。这种方式简单直观，通过在HTML中定义特定的标签和属性，可以初始化EasyUI控件。例如，可以通过在HTML中使用 < table > 标签声明一个数据表格（datagrid）。
JavaScript声明方式：
使用JavaScript代码在页面加载完成后通过EasyUI提供的API来动态创建和初始化UI控件。这种方式更加灵活，通过在JavaScript中调用EasyUI的API方法，可以在运行时动态生成UI控件。通常，这种方式用于需要更多动态性和交互性的场景。
这两种声明方式根据项目的需求选择使用。HTML标记声明方式适用于静态页面和简单场景，而JavaScript声明方式适用于需要在运行时动态生成和控制UI的复杂场景。在实际开发中，开发者可以根据具体的项目要求和UI设计选择适当的声明方式。
